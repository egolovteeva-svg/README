## Идея компании
Сервис по подарочной упаковке товаров из интернет-магазинов (Ozon, Wildberries, Lamoda и тд) с последующей доставкой заказчику.
Продается услуга клиентам маркетплейсов(B2C), которые хотят приобрести готовый подарок "под ключ".
Клиент оформляет заказ на маркетплейсе до места упаковки(пвз), WrapItUp упаковывает товар в соответствии с согласованным дизайном, после чего товар бережно доставляется курьером.
